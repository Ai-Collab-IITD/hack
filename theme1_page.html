Challenge Overview

AI agents for collaborative content creation and knowledge curation

This hackathon case centres on leveraging the power of AI agents for content generation and knowledge curation in a collaborative environment. AI agent can act as starting coach for knowledge curation topics, doing the task assignment to participants, providing tools for exploring more data on topics.

This AI agent can also expose tools converting information from textual to image formats (e.g. tools like SORA, Microsoft Designer could be leveraged). Not everything has to be developed from scratch ‚Äì rather rely upon prior tools to stitch together in a flow to enable easy content creation.

Such a tool could have usage in writing papers, completing assignments, generating social media content etc.
